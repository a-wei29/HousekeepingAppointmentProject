# 阿姨准入标准

## 基本条件
- 年龄：25-55周岁
- 学历：初中以上
- 健康：3个月内体检合格[citation:4]
- 背景：无犯罪记录[citation:7]

## 健康要求
必须提供以下体检合格证明：
- 常规体检
- 传染病筛查（乙肝、结核等）
- 皮肤病检查
- 精神健康评估[citation:7]

## 技能要求
根据不同岗位设置不同技能等级[citation:8]

### 初级家政员
- 基础清洁技能
- 普通衣物洗涤
- 家常菜制作
- 服务礼仪规范

### 中级家政员
- 深度清洁技能
- 分类洗涤熨烫
- 特色菜制作
- 简单急救知识

### 高级家政员
- 特殊材质养护
- 家庭宴会准备
- 儿童早教能力
- 老人护理知识
