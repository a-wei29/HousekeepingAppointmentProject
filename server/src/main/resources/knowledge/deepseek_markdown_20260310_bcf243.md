# 附加费用说明

## 节假日加价
- 国家法定节假日：基础价格×1.5倍
- 春节（初一至初三）：基础价格×2倍

## 特殊服务加价[citation:7]
| 服务类型 | 加价标准 | 说明 |
|---------|---------|------|
| 高空作业 | +50元/次 | 3层以上外窗清洁 |
| 重油污处理 | +30元/次 | 长期未清洁油烟机 |
| 贵重物品保养 | 按物品议价 | 实木地板、真皮沙发 |
| 超面积服务 | +30元/50平米 | 超过150平米部分 |

## 交通费用
- 五环内：免费
- 五环外：按距离收取10-30元
- 远郊区县：按实际交通费收取

## 其他费用
- 材料费：清洁剂、垃圾袋等已包含
- 设备费：吸尘器、蒸汽机等由公司提供
- 如需特殊材料（如进口清洁剂），实报实销
