# 服务过程标准规范

## 一、到岗签到[citation:5]
- 提前10分钟到达服务地点
- 整理着装、佩戴工牌
- 电话联系客户确认入户
- 穿鞋套入内

## 二、服务前沟通
- 与客户确认今日服务内容
- 了解特殊需求或注意事项
- 询问贵重物品存放位置[citation:7]
- 确认工具材料摆放位置

## 三、服务执行规范
### 清洁类服务
- 从上到下，从里到外
- 干湿分离，分类清洁
- 重点区域（厨房、卫生间）单独处理
- 移动物品归位

### 照护类服务
- 先观察后行动
- 与被照护人保持沟通
- 记录异常情况
- 按时喂药/喂食

## 四、服务验收
- 服务结束后请客户验收
- 对照服务清单逐项确认
- 如有不满意立即返工
- 填写服务记录单[citation:5]

## 五、离开现场
- 清理服务产生的垃圾
- 检查物品是否归位
- 确认水电气关闭
- 与客户道别
