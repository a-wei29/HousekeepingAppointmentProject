# 投诉处理流程

## 投诉渠道
- 客服电话：400-xxx-xxxx（24小时）
- 微信公众号在线客服
- APP在线投诉
- 服务评价系统

## 处理时效[citation:2]
- 普通投诉：24小时内响应，3个工作日内解决
- 紧急投诉：2小时内响应，24小时内解决
- 严重投诉：立即启动应急预案

## 投诉分类处理

### 服务质量问题
1. 核实情况（查看服务记录、回访双方）
2. 如属实，安排免费返工或部分退款
3. 对阿姨进行再培训
4. 3个工作日内回访客户满意度

### 人员更换要求[citation:4]
- 试用期内可无条件更换（3个工作日）
- 正式服务期：出现以下情况可要求更换
  - 阿姨有违法行为
  - 患有传染性疾病
  - 服务态度恶劣
  - 技能不达标

### 退费问题
- 单次服务不满意：48小时内申请，核实后退款
- 预付卡退款：按合同约定执行，余额需在15日内退还[citation:7]

## 投诉升级处理
如客户对处理结果不满意，可申请：
1. 客服主管介入
2. 区域经理协调
3. 第三方调解（消协、行业协会）

## 投诉记录与分析
- 每月统计投诉类型分布
- 分析高频问题根源
- 优化服务流程和培训内容
- 持续改进服务质量[citation:5]
