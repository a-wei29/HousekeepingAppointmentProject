# 育儿嫂服务介绍

## 服务对象
0-3岁婴幼儿

## 服务内容
- 婴幼儿生活照料：喂奶、换尿布、洗澡、抚触
- 辅食制作：按月龄科学添加辅食
- 早期教育：讲故事、听音乐、认知训练
- 户外活动：每天不少于1小时户外活动
- 健康监测：体温、便便、睡眠记录

## 阿姨资质要求
- 持有育婴师职业资格证书
- 3年以上工作经验
- 健康证（3个月内）
- 无犯罪记录证明[citation:7]

## 服务时间
- 白班：8:00-18:00
- 住家：24小时（每月休息4天）[citation:4]

## 价格说明
- 初级育儿嫂：100-120元/小时
- 中级育儿嫂：120-140元/小时
- 高级育儿嫂：140-160元/小时
