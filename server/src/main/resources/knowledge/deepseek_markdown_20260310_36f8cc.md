# 家政服务消费提示

根据市场监管部门和消协发布的消费警示，请消费者注意以下事项[citation:2][citation:7]：

## 一、选择正规公司
- 查看营业执照等经营资质
- 通过"家政信用查APP"查询企业信用[citation:7]
- 优先选择行业协会推荐企业

## 二、核实人员信息
- 扫描信用二维码，查看：
  - 有效身份证明
  - 5年内有无犯罪记录[citation:7]
  - 有无严重精神病史
- 查看健康证明（3个月内）
- 查看职业技能培训证明

## 三、签订正式合同[citation:4]
- 明确服务内容、标准、价格
- 约定违约责任
- 明确退费规则
- 不轻信口头承诺

## 四、预付卡消费警示[citation:2]
- 记名卡限额≤5000元
- 不记名卡限额≤1000元
- 警惕"充值越多优惠越大"的诱导
- 保留付款凭证

## 五、维权渠道
发生纠纷时：
1. 与商家协商
2. 拨打12345投诉
3. 登录全国消协智慧315平台
4. 向消费者协会求助[citation:2]
