# 保洁服务介绍

## 服务类型
### 日常保洁
- 全屋表面清洁：家具表面、地面、台面
- 厨房清洁：灶台、水槽、台面
- 卫生间清洁：马桶、淋浴房、洗手台

### 深度保洁
包含日常保洁所有内容，额外增加：
- 抽油烟机清洗（外部+滤网）
- 窗户玻璃清洗（内面）
- 纱窗清洗
- 墙面/天花板除尘
- 冰箱/烤箱内部清洁

### 开荒保洁
适用于新房装修后、长期无人居住房屋
- 装修残留物清理
- 全屋全方位深度清洁
- 玻璃内外清洗
- 需提前预约（至少提前3天）

## 注意事项
- 深度保洁和开荒保洁需提前说明房屋面积
- 复式、别墅等特殊户型价格另议
- 如有特殊材质（大理石、实木）需提前告知[citation:7]
